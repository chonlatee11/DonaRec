## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-19 | Tampering / Info Disclosure | admin template injection via preview/pdf | high | mitigate | Same sandboxed pipeline as production (pdf.RenderPDF): JS-disabled + network-blocked + SetDocumentContent; sample-data only (D-58/D-61) |
| T-04-20 | Tampering | donor/sample field breaking out of HTML | high | mitigate | html/template contextual autoescaping; template.Parse validation on save |
| T-04-21 | Tampering | malicious brand-image upload | high | mitigate | Magic-byte validation (image/jpeg+png only) + 2MB cap, reusing the proven validateSlip pattern |
| T-04-22 | Elevation of Privilege | unauthorized config change | high | mitigate | adminGroup RequireRoles(RoleAdmin); non-Admin → 403 (E2E-verified, Conventions gate) |
| T-04-23 | Repudiation | silent config change | medium | mitigate | Append-only audit (AppendAuditEntryTx) on every save (D-58) |
